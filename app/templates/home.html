User Instructions


Get a time estimate:

Select the "Input" tab. Type data into the corresponding box using the specified units, then click "Save Data." Navigate to the "Prediction" page.


View or change saved data:

Navigate to the "Search Data" tab. Locate the correct project (projects are listed in reverse chronological order), then select "View/Edit." Make the edits, then select "Save Data."


To log in/log out:

Log in by selecting the "Login" tab and entering credentials. Log out by selecting the "Logout" tab in the same location.


Revisit instructions after navigating away:

Select the Midwest Solar logo in the top left to return to the home page and revisit these instructions.


Admin Instructions


Add or remove users:

Navigate to the "Admin Portal" tab, then click on "Admin Portal" under "Access Controls."


Model training page:

Navigate to the "Admin Portal" tab, then select "Train Model."

Upload data file:

Navigate to the "Admin Portal" tab, then select "Upload File."


Change input page:

Navigate to the "Admin Portal" tab, then select "Edit Input Field."
